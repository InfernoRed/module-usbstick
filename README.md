# USB Stick Module

This web server component provides a web interface to simple USB stick
operations on linux.

## Dependencies

```
sudo apt-get update && sudo apt-get install -y --no-recommends pmount
```

## Install Requisite Software

```
yarn install
```

## Run Tests

```
yarn test
```

## Start the Server

```
yarn start
```
